Involvex AI — Settings

Keys are stored locally on this device only (chrome.storage.local) and sent directly to the provider you choose. No Involvex server is involved.

Google Gemini

Get a key at aistudio.google.com/apikey

OpenAI

Get a key at platform.openai.com/api-keys

Anthropic Claude

Get a key at console.anthropic.com

OpenRouter

One key, hundreds of models (incl. free ones). Get a key at openrouter.ai/keys

OpenCode Zen

OpenAI-compatible. Free models end with -free, e.g. mimo-v2.5-free. Use Load models for the full live list.

Custom (OpenAI-compatible)

For OpenCode, Kilo Code, LM Studio, vLLM, LiteLLM, or any server that exposes /v1/chat/completions.

Local (Ollama)

Run a small local model, e.g. ollama pull qwen2.5:3b. Start with OLLAMA_ORIGINS=* ollama serve so the extension can reach it.

Backup & Sync

Back up your bookmarks, extension settings and installed-extension list to a private GitHub Gist. Serverless and free — no Involvex or third-party server. API keys are never included in a backup.

Create a token at github.com/settings/tokens → Fine-grained or classic with only the gist scope. Restore adds bookmarks into a new "Involvex Restore" folder (it never overwrites existing bookmarks).
